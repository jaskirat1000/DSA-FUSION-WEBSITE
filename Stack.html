Stack

Learn about stacks and their LIFO principle

What is a Stack?

A stack is a linear data structure that follows the Last In First Out (LIFO) principle. The element inserted last is accessed first.

Key Characteristics:

  • LIFO Principle: Last In First Out
  • Two Main Operations: Push (add) and Pop (remove)
  • Top Pointer: Always points to the topmost element
  • Implementation: Can be implemented using arrays or linked lists

Basic Operations:

Push

Adds an element to the top of the stack

function push(item) {
  if (top >= capacity-1) {
    console.log("Stack Overflow");
    return;
  }
  stack[++top] = item;
}

Pop

Removes the top element from the stack

function pop() {
  if (top < 0) {
    console.log("Stack Underflow");
    return;
  }
  return stack[top--];
}

Peek

Returns the top element without removing it

function peek() {
  if (top < 0) {
    console.log("Stack is empty");
    return;
  }
  return stack[top];
}

isEmpty

Checks if the stack is empty

function isEmpty() {
  return top < 0;
}

Arrays

Learn about arrays and their applications in programming

What is an Array?

An array is a collection of elements of the same type, stored in contiguous memory locations. Each element can be accessed directly by its index.

Key Characteristics:

  • Fixed Size: The size of an array is fixed at the time of declaration.
  • Contiguous Memory: Elements are stored in adjacent memory locations.
  • Random Access: Any element can be accessed in constant time using its index.
  • Homogeneous Elements: All elements must be of the same data type.

Basic Operations:

Traversal

Visiting each element of the array exactly once.

for(int i=0; i<n; i++) {
  print(arr[i]);
}

Insertion

Adding an element at a specific position.

// Insert x at position pos
for(int i=n; i>pos; i--) {
  arr[i] = arr[i-1];
}
arr[pos] = x;
n++;

Deletion

Removing an element from a specific position.

// Delete element at position pos
for(int i=pos; i<n-1; i++) {
  arr[i] = arr[i+1];
}
n--;

Searching

Finding the position of an element.

// Linear Search
for(int i=0; i<n; i++) {
  if(arr[i] == key) return i;
}
return -1;

Time Complexity:

Operation Time Complexity
Access O(1)
Search O(n)
Insertion O(n)
Deletion O(n)

Practice Problems

1. Reverse an Array

Write a function to reverse the elements of an array in-place.

void reverseArray(int arr[], int n) {
  int start = 0, end = n-1;
  while(start < end) {
    int temp = arr[start];
    arr[start] = arr[end];
    arr[end] = temp;
    start++;
    end--;
  }
}

2. Find Maximum Subarray Sum (Kadane's Algorithm)

Given an array of integers, find the contiguous subarray with the largest sum.

int maxSubarraySum(int arr[], int n) {
  int max_current = arr[0];
  int max_global = arr[0];
  
  for(int i=1; i<n; i++) {
    max_current = max(arr[i], max_current + arr[i]);
    if(max_current > max_global) {
      max_global = max_current;
    }
  }
  return max_global;
}

3. Two Sum Problem

Given an array of integers, return indices of the two numbers that add up to a specific target.

vector<int> twoSum(vector<int>& nums, int target) {
  unordered_map<int, int> map;
  for(int i=0; i<nums.size(); i++) {
    int complement = target - nums[i];
    if(map.find(complement) != map.end()) {
      return {map[complement], i};
    }
    map[nums[i]] = i;
  }
  return {};
}
